Sphinx操作メモ¶

windows環境。conf.pyにプロジェクトの設定。Windowsでもパスの区切りは/を使用

インストール¶

Pythonのインストール、環境変数の設定をする。 Pythonのインストール先へ移動:

cd %USERPROFILE%\AppData\Local\Programs\Python\Python36-32

Sphinx用のvenv環境をユーザディレクトリ配下にmysphinxという名前で作成:

python -m venv %USERPROFILE%\mysphinx

仮想環境に入る:

%USERPROFILE%\mysphinx\Scripts\activate

pipコマンドでインストール:

pip install Sphinx

新しいプロジェクトの作成:

sphinx-quickstart

インストール時に色々聞かれるので`ドキュメント <http://sphinx-users.jp/gettingstarted/sphinxquickstart.html>`_ をみながら入力。

普段の操作¶

コマンドプロンプトから仮想環境に入り作業フォルダへ移動:

%USERPROFILE%\mysphinx\Scripts\activate
cd 作業フォルダ

#出力先(デフォルトは_build)の中身を一度全部消してhtml出力:

make clean
make html

出力先はMakefile(とmake.bat)に記述している。

設定¶

favicon¶

ペイントで32×32の24ビットビットマップを作成 conf.pyに以下の記述追加 html_title = '昨日の豆腐 tofulog' html_favicon = 'img/favicon.ico'

github用のhtml¶

conf.pyのextensionに”sphinx.ext.githubpages”を追加する。 html生成時に.nojekyllを自動生成してくれる。

build時の読み取り除外¶

conf.pyの exclude_pattern にプロジェクトフォルダからの相対パスを指定

文法¶

同じ段落のすべての行は、インデントを同じ高さにそろえて、左揃えにしなければならない。 以下のサイトに基本的な文法が記載。 サイト1 サイト2

外部にあるcsvファイルからデータを読み込む¶

--file-insertion-enabled=disable　で無効化されているかもしれない :file:を:url:にすると外部ファイルが読み込める

画像¶

alignは”top”, “middle”, “bottom”, “left”, “center”, “right”が使える

数式¶

http://www.sphinx-doc.org/ja/stable/ext/math.html conf.pyのextensionに”sphinx.ext.mathjax”を追加する。(デフォルトで追加されているかもしれない) ブラウザ側でjavascriptで数式をレンダリングする。 mathjaxの替わりにsphinx.ext.pngmathを設定してpng画像を生成しておくこともできるがlatexやdvipngなどのプログラムのインストールが必要。

表記方法:

.. math::
  F = ma

インライン表記:

:math:`a^2 + b^2 = c^2`
